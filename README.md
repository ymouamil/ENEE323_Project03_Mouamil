# ENEE 323 DSP Project
## Fourth-Order IIR Filter Analysis

This project analyzes a fourth-order IIR filter including:

- Impulse response behavior
- Pole-zero analysis
- Stability (BIBO)
- Closed-loop feedback system
- Gain selection for stability

## Files
- MATLAB script: `iir_feedback_project.m`
- Figures: `/figures`
- LaTeX report: `report.tex`

## Results
- Open-loop system is unstable
- Closed-loop system becomes stable for K ≈ 0.451
